# Kitchen Management System

A web-based kitchen management system designed to streamline order handling between customers, kitchen staff, and administrators.

## Features
- Real-time order receiving
- Order status tracking (Pending / Cooking / Ready)
- Kitchen dashboard optimized for fast operation
- Role-based access control (Admin / Kitchen Staff)

## Tech Stack
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: Express / PHP
- Database: Firebase

## My Role
- Designed system workflow and architecture
- Implemented order processing logic
- Developed role-based access control
- Optimized UI flow for real kitchen usage

## Use Case
- Restaurants and food courts
- Reduces miscommunication between front and kitchen
- Improves order processing efficiency
